导入

```javascript
import { env } from 'xun-util';
```

### env.isDev()

是否是开发环境。

### env.isProd()

是否是生产环境。

### env.isWechat()

是否是微信环境。

### env.isAndroid()

是否是安卓设备。

### env.isIphone()

是否是Iphone设备。

### env.isMobile()

是否是移动端。

### env.isIpad()

是否是Ipad设备。

### env.isSafari()

是否是Safari浏览器。

### env.isBrowser()

是否是浏览器环境。

### env.isIE()

是否是IE浏览器。

### env.isIE9()

是否是IE9浏览器。

### env.isIOS()

是否是IOS设备。

### env.isEdge()

是否是Edge浏览器。

### env.isChrome()

是否是Chrome浏览器。
