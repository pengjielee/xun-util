导入

```javascript
import { type } from 'xun-util';
```

### type.isArray()

是否是数组。

### type.isString()

是否是字符串。

### type.isPlainObject()

是否是Object。

### type.isRegExp()

是否是正则。


### type.isDate()

是否是日期。


### type.isFunction()

是否是函数。

### type.isNumber()

是否是Number。

### type.isBoolean()

是否是Boolean。

### type.isSymbol()

是否是symbol。

### type.isStatic()

是否是除symbol外的原始数据。

### type.isPrimitive()

是否是基本类型。

### type.isUndefined()

是否是undefined。


### type.isNull()

是否是null。

### type.isArrayLike()

是否是类数组。

### type.isPromiseLike()

是否是类Promise。

### type.isObjectLike()

是否是类对象。

### type.isValidJSON()

是否是合法的JSON。
